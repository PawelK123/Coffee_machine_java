Coffee Machine (Java)

A simple console-based coffee machine simulation built in Java using Object-Oriented Programming (OOP) principles.

Description:

This program allows the user to interact with a virtual coffee machine. Users can prepare different types of coffee or refill ingredients when needed.

The application continuously checks whether there are enough ingredients to complete the selected action and provides feedback if something is missing.

Features: 
Make different types of drinks:
- Espresso
- Americano
- Cappuccino
- Hot Milk
- Refill ingredients (water, milk, coffee beans)
Automatic ingredient checking before making a drink
Clear console messages for user interaction
Technologies:
Java
Object-Oriented Programming (OOP)
